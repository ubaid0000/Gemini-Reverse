## 📖 What is Gemini-Reverse?

Gemini-Reverse is a simple program for Windows that lets you use Gemini’s chat and features. It connects to gemini.google.com and works like a helper for chatting, sharing files, and more. You do not need to know programming or code to use it. It works quietly in the background and shows you your chats and files.

This program supports:

- Live chat with streaming replies  
- Sending and receiving messages in sessions  
- Sharing small files easily  
- Using special Gemini gems (small data items)  
- Safe use on Windows machines  

Gemini-Reverse is built with Node.js and TypeScript but does not require you to install or run those yourself. It comes ready to use.
